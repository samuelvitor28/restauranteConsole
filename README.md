# 🍽️ Restaurante no Console

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=java&logoColor=white)

Aplicação desenvolvida em **Java** que simula o funcionamento de um restaurante diretamente no console.  
O objetivo do projeto é praticar conceitos de **programação orientada a objetos (POO)**, organização de código e lógica de negócio.

---

## 📌 Funcionalidades

- Cadastro de pratos
- Listagem de pratos
- Organização por categorias
- Simulação de operações básicas de um restaurante
- Execução totalmente via terminal

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Classes e interfaces
- Organização em camadas (ex: modelo, serviço)
- Manipulação de listas

---

## ⚙️ Tecnologias

- Java
